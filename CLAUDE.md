# Overview

This repository mainly contains the jql suite of tools. jql is a JSON backed personal database written in go. Think of it like a spreadsheet for words, a modern version of Lotus Agenda. It is json backed to allow for git-based version control and visibility into changes. It has a variety of use-cases, the most prominent of which is the timedb. Timedb is mean to be "second brain" or secondary storage for your brain. It has many functions centered around time management including project planning, habit tracking, and journaling. Jql also supports macros which can read and edit data on user-triggered events. These are typically written in python.
